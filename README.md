# Sigma Marketplace

> B2B Field Service Marketplace — connecting clients with managed technician workforces for structured, SOP-driven site work.

---

## Overview

Sigma Marketplace is a B2B platform that enables organizations to dispatch field technicians (network engineers, IT support, infrastructure teams) to client sites. Technicians execute work based on predefined Standard Operating Procedures (SOPs), mark tasks with photo evidence, and get compensated per completed task.
